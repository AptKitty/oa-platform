■ 项目概况
  项目名称：企业协同办公与流程审批系统（OA平台）
  作者：陈健聪  学号：2531020130218
  技术栈：Java 8 + Swing + MyBatis + MySQL 8.0 + Maven
  规模：97个Java源文件 / 20个Mapper XML / 33张数据库表 / 26个Swing面板

■ 角色设计（5种）
  ┌──────────┬──────────────────────────────────┐
  │ 超级管理员 │ 组织架构维护、角色权限分配、系统审计  │
  │ 部门主管  │ 审批下级申请、部门人员管理           │
  │ 普通员工  │ 发起申请、考勤打卡、查看公告         │
  │ HR       │ 考勤统计、员工档案、假期管理          │
  │ 行政     │ 会议室/资产/用车管理、公告发布        │
  └──────────┴──────────────────────────────────┘

■ 核心功能模块（7大模块）
  1. 组织架构与权限管理（RBAC）
  2. 可配置流程审批引擎（核心难点）★
  3. 考勤管理子系统
  4. 公告与消息通知
  5. 日程与任务管理
  6. 行政管理（会议室/资产/用车）
  7. 数据统计大屏
