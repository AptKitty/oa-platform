============================================================
一、项目概述
============================================================

【项目名称】企业协同办公与流程审批系统（OA平台）
【项目类型】Java Swing 桌面应用程序
【作者】陈健聪  学号：2531020130218

【项目目标】
  构建一套完整的企业协同办公平台，核心是可配置的流程审批引擎，
  支持会签/或签/条件分支等多种审批模式，配合考勤、公告、日程、
  行政、统计等辅助模块，形成完整的OA办公生态。

【角色设计（5种）】
  ┌──────────┬──────────────────────────────────┐
  │ 超级管理员 │ 组织架构维护、角色权限分配、系统审计  │
  │ 部门主管  │ 审批下级申请、部门人员管理           │
  │ 普通员工  │ 发起申请、考勤打卡、查看公告         │
  │ HR       │ 考勤统计、员工档案、假期管理          │
  │ 行政     │ 会议室/资产/用车管理、公告发布        │
  └──────────┴──────────────────────────────────┘

【核心功能模块（7大模块）】
  1. 组织架构与权限管理（RBAC）
  2. 可配置流程审批引擎（核心难点）★
  3. 考勤管理子系统
  4. 公告与消息通知
  5. 日程与任务管理
  6. 行政管理（会议室/资产/用车）
  7. 数据统计大屏

============================================================
二、系统架构设计
============================================================

【技术架构分层】
  ┌────────────────────────────────────────────┐
  │         表现层 (Swing GUI)                  │
  │  LoginFrame / MainFrame / 26个Panel         │
  ├────────────────────────────────────────────┤
  │         业务逻辑层 (Service)                │
  │  WorkflowService / UserService / ...       │
  ├────────────────────────────────────────────┤
  │         数据访问层 (DAO + Mapper XML)       │
  │  20个Mapper XML，参数化查询防SQL注入        │
  ├────────────────────────────────────────────┤
  │         数据库层 (MySQL 8.0)                │
  │  oa_platform 数据库，33张表                 │
  └────────────────────────────────────────────┘

【调用链路】
  Panel (用户点击按钮)
    → Service (处理业务逻辑、事务管理)
      → DAO接口 (MyBatis动态代理)
        → Mapper XML (执行SQL)
          → MySQL 数据库

【设计模式】
  - 分层架构：表现层/业务层/数据层严格分离
  - 单例模式：MyBatisUtil（SqlSessionFactory全局唯一）
  - ThreadLocal线程绑定：当前用户信息、数据库会话复用
  - 模板方法模式：BasePanel定义公共界面组件和工具方法
  - 工厂模式：ApplyPanel根据字段类型动态创建Swing组件

四、核心模块详细设计
============================================================

【模块1：流程审批引擎】★ 最复杂模块

  4.1 审批模板管理
    - wf_form_template：模板基本信息（名称、编码、分类）
    - wf_form_field：模板字段定义
    - 支持6种字段类型：
      TEXT(文本框) / NUMBER(数字) / DATE(日期)
      / SELECT(下拉) / TEXTAREA(多行文本) / ATTACHMENT(附件)
    - 前端动态渲染：根据字段类型自动生成对应Swing组件

  4.2 审批链配置（wf_process_node的5种节点类型）
    ┌───────────┬──────────────────────────────────┐
    │ APPROVE   │ 单人审批：一人通过即前进到下一节点│
    │ SIGN      │ 会签：多人并行，全部通过才前进    │
    │ OR_SIGN   │ 或签：多人并行，任一人通过即前进  │
    │ CC        │ 抄送：自动记录后跳过，不需人审批  │
    │ CONDITION │ 条件分支：根据表单数据自动路由    │
    └───────────┴──────────────────────────────────┘

  4.3 审批人解析（3种方式）
    - SPECIFIC_USER：指定具体某个用户
    - DEPT_LEADER：动态查找申请人所属部门的负责人
    - ROLE：查找拥有指定角色的所有用户

  4.4 审批状态机
    DRAFT（草稿）→ PENDING（待审批）→ APPROVING（审批中）
      → PASSED（通过）或 REJECTED（驳回）或 CANCELLED（撤销）

  4.5 事务安全
    - approve/reject/submitProcess 使用手动事务
    - openSession(false) → 多表写入 → commit 或 rollback
    - 保证数据一致性：不会出现"任务更新了但记录没写"的情况

【模块2：RBAC权限模型】

  - 用户(User) → 中间表(sys_user_role) → 角色(Role)
  - 角色(Role) → 中间表(sys_role_menu) → 菜单(Menu)
  - 菜单类型：MENU(菜单项) / BUTTON(按钮级权限)
  - 5种预设角色，登录后动态加载不同侧边栏

【模块3：考勤管理】

  - 上下班打卡（记录时间+IP地址）
  - 迟到判定：上班时间 > 9:00
  - 早退判定：下班时间 < 18:00
  - 旷工判定：当天无任何打卡记录
  - 请假申请关联审批流程，通过后自动扣减额度
  - 月度考勤汇总表，支持Excel导出

【模块4：公告与消息】

  - 公告：富文本HTML编辑、附件上传、定时发布、置顶
  - 已读/未读追踪：精确到人，谁看了谁没看
  - 站内消息：审批通知、系统通知、已读标记

【模块5：日程与任务】

  - 日历视图：月/周/日三种视图切换
  - 会议管理：发起会议→选择参与人→时间冲突检测→发送通知
  - 任务分配：主管向下属分配任务，设置截止日期，跟踪完成状态

【模块6：行政管理】

  - 资产管理：登记→领用→归还→报废，完整操作历史
  - 会议室管理：CRUD + 预约 + 时段冲突检测
  - 用车管理：车辆信息 + 出车/归队记录

【模块7：统计大屏】

  - 概览卡片：用户总数、部门数、待审批数、今日提交数
  - 审批效率排名柱状图（JFreeChart）
  - 各部门出勤对比柱状图
  - 请假类型分布饼图
  - 待办事项实时统计

八、部署说明
============================================================

【环境要求】
  - JDK 8 或更高版本
  - MySQL 8.0+（root密码设为root）
  - 操作系统：Windows / Linux / Mac

【部署步骤（4步）】
  第1步：安装并启动MySQL服务
  第2步：执行"数据库建表.sql"创建 oa_platform 数据库和33张表
  第3步：执行"测试数据.sql"导入测试数据
  第4步：双击"启动系统.bat"运行程序
         （或命令行：java -jar OA办公系统.jar）

【注意事项】
  - MySQL密码不是root → 修改 mybatis-config.xml 中的 password
  - MySQL端口不是3306 → 修改 mybatis-config.xml 中的 url
  - 首次启动稍慢（初始化数据库连接池）

【发布包文件清单】
  OA办公系统.jar        (31.4 MB, 包含所有依赖)
  数据库建表.sql         (33张表DDL)
  测试数据.sql           (18用户+全部测试数据)
  mybatis-config.xml     (数据库连接配置)
  启动系统.bat           (Windows双击启动)
  设计说明书.txt         (本文档)
  README.txt             (快速使用说明)
